# Servers (Monorepo)

## TechStacks

- Graphql
- Apollo Client
- Prisma
- MongoDb

## Scripts

### Run Services

```bash
yarn start:dev <service-name>
```

### Build Services

```bash
yarn build <service-name>
```

### Prisma

```bash
npx prisma format
```

```bash
npx prisma generate
```

```bash
npx prisma db push
```

```bash
npx prisma db studio
```
