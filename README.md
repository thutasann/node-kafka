# Nodejs Kafka Microservices with Elastic Search

This is the tiny microservices with Kafka and elastic search from scratch.

> This repo is mainly focused on Backend.

## Tech Stacks

- Nodejs
- Kafka
- Elastic Search
- Prisma
- Graphql
- Mysql
- Nextjs

## Folder Structure

```bash
└───node-kafka
  |    ├───app-backend
  |    |   ├───catalog_service
  |    |   ├───order_service
  |    ├───app-frontend
```
